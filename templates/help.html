IDEAS: Image DEep Annotator for Segmentation

How to use the editor:

+ Customize the classes and objects

+ Fundamental annotation pipeline

+ Auxiliary functions to faster the annotation process

    -  lock/unlock an object

    - get more precise label by giving more interactions.

    - opacity controlling to check label accuracy.

    - 'Rectangle' tool to choose a local bounding box the wanted object is in, back-end algorithm will analysis on this local window instead of the whole image.
